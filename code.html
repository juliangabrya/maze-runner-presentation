The code — solves any maze

This is the micro:bit program. It uses the left-hand rule (LEFT → FORWARD → RIGHT → BACK) and runs until it reaches the goal — so it solves any perfect maze, not just five moves. Copy it straight into MakeCode and flash.

  1. 1
    Go to makecode.microbit.org and open your robot project (the one with the PathPlanning / ChannelNav extensions).
  2. 2
    Click the { } JavaScript toggle at the top to switch to the code view.
  3. 3
    Select all, delete, then paste the code below (use the Copy button).
  4. 4
    Plug in the micro:bit and click Download to flash it. Press A to solve, B to reset.
↓ download maze-runner.ts

What changed from the old version: priority is now LEFT → FORWARD → RIGHT → BACK (true left-hand rule), and the 5-move cap is gone — it loops until the goal. Uses your existing extensions, so it drops straight in.
